# LangChain 요약 학습 Agent 안내

## 역할

이 폴더는 4개의 Jupyter Notebook을 초보자도 다시 보기 쉽게 정리한 학습용 요약 공간입니다.

Agent는 다음 원칙으로 자료를 설명합니다.

- 어려운 용어는 먼저 쉬운 말로 바꿔 설명합니다.
- 한 번에 너무 많은 내용을 설명하지 않습니다.
- 개념 → 단계 → 예시 → 체크포인트 순서로 설명합니다.
- 원본 노트북의 실습 흐름은 유지하되, 복습하기 쉽게 짧게 정리합니다.

## 폴더 구성

```text
랭체인요약/
  01_1일차_LangChain_기초/
    README.md
  02_2일차_RAG와_Streamlit/
    README.md
  03_MCP_기초/
    README.md
  04_Modular_RAG와_LangGraph/
    README.md
  agent.md
```

## 추천 학습 순서

1. `01_1일차_LangChain_기초`에서 LangChain, Prompt, LCEL, Memory를 먼저 봅니다.
2. `02_2일차_RAG와_Streamlit`에서 문서 기반 챗봇과 웹 UI 흐름을 봅니다.
3. `03_MCP_기초`에서 LLM이 외부 도구를 실행하는 구조를 봅니다.
4. `04_Modular_RAG와_LangGraph`에서 복잡한 RAG 흐름을 그래프로 설계하는 방법을 봅니다.

## 설명 방식

각 요약 파일은 아래 형식을 따릅니다.

- 한 줄 요약: 전체 주제를 가장 쉽게 설명합니다.
- 핵심 개념: 반드시 알아야 할 용어만 모읍니다.
- 단계별 정리: 실습 흐름을 몇 단계로 나눕니다.
- 쉬운 예시: 실제 상황에 빗대어 이해합니다.
- 체크포인트: 복습할 때 확인할 질문입니다.

## 복습 질문 예시

- LCEL에서 `|` 기호는 어떤 역할을 하나요?
- RAG는 왜 문서를 작은 청크로 나누나요?
- MCP에서 Host, Client, Server는 각각 무슨 역할인가요?
- LangGraph에서 State와 Node는 어떻게 다르나요?

## 주의사항

- `.env` 파일과 API 키는 외부에 공유하지 않습니다.
- 원본 `.ipynb` 파일은 실습용, 이 폴더의 `README.md` 파일은 복습용으로 사용합니다.
- 코드 예시는 이해를 돕기 위한 샘플이므로 실제 실행 전 패키지 설치와 API 키 설정을 확인합니다.
